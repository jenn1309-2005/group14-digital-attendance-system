# Attendance Management System

## 📌 Project Overview
This system is designed to help lecturers and students manage attendance digitally. It allows lecturers to mark attendance, and students to view their attendance records easily.

## 🎯 Features
- Student login using Student Number
- Lecturer login using Staff ID
- Mark attendance per course
- View attendance records
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Programming Language: (e.g. Java / PHP / Python)
- Database: (e.g. MySQL / SQLite)
- Frontend: HTML, CSS, JavaScript (if applicable)

## 🚀 How to Run the System
1. Download or clone the repository
2. Open the project in your IDE (NetBeans / VS Code / Eclipse)
3. Import the database file (if included)
4. Run the main file (e.g. Main.java / index.php / app.py)
5. Login using test credentials

## 👤 User Roles
- Lecturer: Can mark and manage attendance
- Student: Can view attendance records
